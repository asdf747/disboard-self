## Disboard Self
- Just a selfbot to bump you discord server.
## How Use:
> Configure the config.js file
```js
module.exports = {
  token: 'YOU SELFBOT TOKEN',
  guild: 'GUILD ID',
  channel: 'CHANNEL ID'
}
```
> After this, just execute the commands:
```
npm install
```
or
```
yarn
```
and
```
node .
```
